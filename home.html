Zomato Clone

The food delivery web application is a comprehensive full-stack platform built using HTML, CSS, Bootstrap, React, Node.js, Express.js, and MongoDB. Users can select their city from a dropdown list to view restaurant names and addresses. The app offers meal selection, restaurant filtering by cost and cuisine, along with restaurant details including ratings, contact information, and menus. Users can add items to the cart, place orders, and view their orders. Authentication via login and signup ensures secure access to order history and cancellation for each user.

INotebook

The iNotebook web application is a full-stack solution developed using HTML, CSS, Bootstrap, React, Node.js, Express.js, and MongoDB. Users can securely store and manage their notes by providing titles, descriptions, and tags. The application features authentication with login and signup functionality, ensuring users can access and update their own notes. Additionally, the app offers a dark and light theme toggle for a personalized user experience.
